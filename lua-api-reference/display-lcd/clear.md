# lcd.clear(\[color])

Clear the LCD screen

@status current Introduced in 2.3

### Parameters

* `color_flag` (optional), only on color screens)

### Return value

none

### Avaiable on

* [ ] B\&W LCD radios
* [ ] Grayscale LCD radios
* [x] Color LCD radios

#### Notice

This function only works in stand-alone and telemetry scripts.
